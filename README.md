## 📖 项目简介

**Attention.Fi** 是一个创新的 Web3 平台，将注意力经济与去中心化金融相结合。通过将创作者的社交影响力代币化，并结合预测市场机制，为用户提供一个数据驱动的投资决策平台。

### 核心理念

在注意力稀缺的时代，影响力就是价值。Attention.Fi 让用户可以：
- 🎯 **投资创作者影响力** — 通过 Bonding Curve 机制买卖创作者代币
- 🔮 **参与事件预测** — 在预测市场中对未来事件进行概率定价
- 📊 **数据驱动决策** — 通过丰富的可视化图表洞察市场趋势

---

## ✨ 核心功能

### 1. 创作者市场 (Creator Market)

基于 Twitter 影响力的创作者代币化交易市场。

- **一键发行** — 任何人都可以为 Twitter 创作者发行代币
- **Bonding Curve 定价** — 价格随供需自动调节，早期参与者享受价格优势
- **Attention Score** — 基于粉丝数、互动量等维度的综合影响力评分
- **实时价格图表** — 迷你走势图展示价格变化趋势

### 2. 预测市场 (Prediction Market)

去中心化的事件预测与概率定价平台。

- **多类型事件** — 支持加密货币、政治、体育、科技等多领域预测
- **灵活选项** — 支持二元（Yes/No）及多选项预测市场
- **限价订单** — 支持市价单和限价单，满足不同交易策略
- **概率可视化** — 实时展示各选项的概率走势

### 3. 投资组合 (Portfolio)

全面的资产管理与收益追踪面板。

- **资产分布饼图** — 直观展示投资组合的资产配置
- **净值走势图** — 追踪投资组合的历史表现
- **盈亏分析** — 实时计算持仓盈亏与收益率
- **持仓明细** — 详细展示每个持仓的成本、市值与收益

### 4. 排行榜 (Leaderboard)

发现热门创作者与市场趋势。

- **多维度排序** — 按评分、价格、持有人数、交易量等排序
- **可视化对比** — 条形图直观展示创作者实力对比
- **活动流** — 实时展示平台交易动态
